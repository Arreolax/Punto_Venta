<h2>🚀 Descripción</h2>
<p>
  <strong>Punto de Venta </strong> es un sistema integral de punto de venta desarrollado en Node.JS, orientado a la gestión eficiente de productos, ventas y clientes.
</p>

<p>
  El proyecto incluye un backend robusto con autenticación segura, control de sesiones y gestión de usuarios,
  además de una interfaz amigable para facilitar su uso.
</p>

<hr>

<h2>🧩 Características principales</h2>
<ul>
  <li>Registro y administración de productos</li>
  <li>Control de ventas en tiempo real</li>
  <li>Gestión de clientes</li>
  <li>Sistema de autenticación con roles</li>
  <li>Registro de actividad administrativa</li>
  <li>Interfaz sencilla y funcional</li>
  <li>Sistema de facturacion mediante API</li>
</ul>

<hr>

<h2>🛠️ Tecnologías utilizadas</h2>

<h3>Backend</h3>
<ul>
  <li>Node.js</li>
  <li>Express.js</li>
</ul>

<h3>Base de datos</h3>
<ul>
  <li>MySQL</li>
</ul>

<h3>Autenticación y seguridad</h3>
<ul>
  <li>JSON Web Tokens (JWT)</li>
  <li>Refresh Tokens</li>
  <li>Middleware de roles y permisos</li>
</ul>

<h3>Facturas</h3>
<ul>
  <li>Facturama</li>
</ul>

<hr>

<h2>🧩 Autenticación y API</h2>
<ul>
  <li>Autenticación basada en JWT + Refresh Token</li>
  <li>Middleware de validación de sesiones</li>
  <li>Sistema de roles</li>
  <li>Control de sesiones en base de datos</li>
  <li>Expiración automática por inactividad (2 horas)</li>
</ul>

<h3>🔐 Funcionalidades</h3>
<ul>
  <li>Login / Logout</li>
  <li>Perfil de usuario</li>
  <li>Cambio de contraseña</li>
  <li>Restablecimiento de contraseña (solo admin)</li>
</ul>

<h3>📜 Seguridad</h3>
<ul>
  <li>Registro en <code>activity_logs</code></li>
  <li>Tokens almacenados como hash</li>
</ul>
